# 📝 TaskMaster Pro

A modern and responsive To-Do List application built using **HTML, CSS, and JavaScript**. TaskMaster Pro helps users manage tasks efficiently with priority-based sorting, progress tracking, real-time statistics, and interactive performance analytics.

---

## 🚀 Features

### ✅ Task Management

* Add new tasks with custom titles
* Assign priority levels (Low, Medium, High)
* Set due dates
* Mark tasks as completed
* Delete tasks with smooth animations

### 📌 Smart Priority Sorting

Tasks are automatically organized by:

1. Pending tasks first
2. High priority → Medium priority → Low priority

This ensures important tasks remain visible and actionable.

### 📊 Real-Time Statistics

Track productivity with:

* Completed task count
* Pending task count
* Daily efficiency percentage

### 📈 Interactive Analytics Graph

Built using HTML5 Canvas:

* Visualizes productivity trends
* Displays progress history
* Interactive hover tooltips
* Tracks user actions and completion updates

### 🎯 Progress Tracking

Dynamic progress bar updates automatically based on completed tasks.

### 📅 Date Management

Each task records:

* Creation date
* Due date

Helping users stay organized and meet deadlines.

### 🎨 Modern User Interface

* Premium dark-themed task card
* Soft gradient background
* Responsive layout
* Smooth animations and transitions
* Priority color indicators

### 📱 Responsive Design

Optimized for:

* Desktop
* Tablet
* Mobile devices
